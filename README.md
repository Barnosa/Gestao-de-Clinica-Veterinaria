# 🐾 VetSys Clinic

Sistema de Gestão de Clínica Veterinária desenvolvido para controlar clientes, animais, veterinários, consultas e vacinas.

## 🔗 Acesse o Projeto

**Deploy:** https://gestao-clinica-veterinaria.netlify.app/

## Funcionalidades

* Cadastro de clientes
* Cadastro de animais
* Cadastro de veterinários
* Registro de consultas
* Registro de vacinas
* Edição e exclusão de registros
* Controle dos relacionamentos entre as entidades

## Relacionamentos

* Cliente → Animais (1:N)
* Animal → Consultas (1:N)
* Veterinário → Consultas (1:N)
* Animal → Vacinas (1:N)

## Tecnologias

* HTML
* CSS
* JavaScript
